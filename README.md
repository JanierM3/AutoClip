# AutoClips

Una aplicación para capturar y gestionar clips de pantalla automáticamente.

## Descripción

AutoClips es una herramienta que permite capturar, procesar y almacenar clips de video de forma automática y eficiente.

## Estructura del proyecto

- **src/** - Código fuente
  - **App/** - Interfaz de usuario
  - **Core/** - Funcionalidades principales
  - **Capture/** - Captura de pantalla
  - **Buffer/** - Gestión de buffer
  - **Events/** - Sistema de eventos
  - **ClipProcessing/** - Procesamiento de clips
  - **Storage/** - Almacenamiento
  - **Notifications/** - Notificaciones
  - **Updater/** - Sistema de actualizaciones

- **docs/** - Documentación
- **tests/** - Pruebas
